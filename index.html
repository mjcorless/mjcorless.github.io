Engineering Drawing Tracker Website

As an intern I primarily worked on the EDT website. EDT is a small internal website used in conjunction to Viper Change Management to keep track of drawings for Viper Change Requests. Before the website's intial development Lockheed engineers needed to track their hours worked for each drawing in spreadsheets. The website also enforces a strict process flow for each drawing type. The complex process flows allow engineers to challenge drawings assigned to them, reject a drawing to a previous state, and advance a drawing to the next state.

SolarSPELL

SolarSPELL is a Solar Powered Educational Learning Library. For my first capstone my group implemented software to extract data from the raspberry pi digital libraries with an android device. The purpose of the project was to gain insight in which artifacts in the digital library were being used without interfering with the digital library functions. We developed an API for android devices to extract the usage information from the LAMP stack on the rasperry pi libraries.
